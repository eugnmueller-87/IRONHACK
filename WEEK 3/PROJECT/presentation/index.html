Deep Intelligence Layer

Beyond single-company queries — Hermes builds dossiers, detects macro themes, and delivers a weekly market summary autonomously.

Company knowledge profiles
Living dossier — every signal permanently updates hermes:profile:{slug}: signal counts, urgency breakdown, risk flags, recent history
Auto-enrichment — at 10+ significant signals, Haiku extracts key_products, pricing_notes, risk_summary automatically
Macro theme clustering
Cross-company patterns — Sonnet groups recent signals into macro themes with synthesis paragraphs. Cached 6h.
Example: "5 chip suppliers flagged export control risk — TSMC, ASML, NVIDIA, KLA, Applied Materials" — without naming any of them in the query
Semantic RAG search
Upstash Vector (BAAI/bge-large-en-v1.5, 1024-dim, cosine). "chip export controls" returns signals from ASML, NVIDIA, KLA — no company names needed
Weekly digest — fully autonomous
Every Sunday 18:00 — Sonnet writes a category-by-category market summary with an overall procurement verdict. Stored 30 days.
Auto-delivered — Icarus forwards the digest to Telegram at 18:30. Zero manual input.
Trend memory
Week-over-week drift — weekly cluster snapshots stored in Redis. GET /trends/delta returns NEW / CONTINUING / RESOLVED themes with no LLM in the comparison path.
Supplier watchlist
High-frequency crawl — watchlisted companies get RSS-crawled every 2h instead of 6h. "Watch TSMC closely" via Telegram adds them instantly.
Content generation pipeline
6-step pipeline — Redis signals → KB (brand voice + landscape) → Haiku brief → Sonnet draft → Telegram staging → approval-gated publish
Isolation principle: every feature catches its own exceptions and never blocks the primary write path. A failing digest never stops signal storage. A failing enrichment never breaks a crawl cycle.
