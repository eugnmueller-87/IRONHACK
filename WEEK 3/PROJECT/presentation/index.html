What HERMES delivers

From raw crawl to actionable intelligence — fully automated.

Intelligence Collection
40+ suppliers tracked across AI, semiconductors, cloud, logistics, and consulting — configurable via a single JSON file
SEC EDGAR filings automatically fetched and summarised — earnings surprises, risk disclosures, M&A activity
Live web search via Tavily fills gaps that RSS misses — product launches, pricing changes, leadership moves
Signal Types
RISK OPPORTUNITY FINANCIAL PRODUCT M&A REGULATORY
AI-Powered Processing
Significance scoring — Claude Haiku rates each item 1–10. Only high-signal items surface in briefings
Deduplication — SHA-256 content hashing prevents the same story from being stored twice
Fuzzy company matching — "TSMC" → "taiwan_semiconductor" automatically, no manual mapping required
On-Demand Queries
"What does Hermes have on NVIDIA?" — returns the latest scored signals for any tracked company
"Give me a Hermes briefing" — top 10 significant signals across all suppliers, sorted by date
SpendLens Integration (built)
enrich_vendor_list() — bulk-enriches SpendLens spend data with Hermes risk levels: HIGH / MEDIUM / LOW per vendor
get_risk_flags() — returns only significant HIGH/MEDIUM signals to feed the SpendLens flag engine. Drop-in via hermes_client.py
