## What this lab is about

Taking a working-but-messy Python script and cleaning it up the professional way —
small focused functions, modular design, and error messages that actually tell you
what went wrong and where.

## How to run

```bash
# 1. Install dependencies
pip install -r requirements.txt

# 2. Add your OpenAI key to .env
OPENAI_API_KEY=sk-...

# 3. Run
python product_generator_refactored.py
```

Output is saved to `results.json`.
